Project Overview
This portfolio was created as part of the Digital Literacy course. The purpose of this project is to demonstrate essential digital skills required for students, including:

creating professional online profiles

understanding cyber safety practices

developing professional communication skills

using online collaboration tools effectively 

Through this project, i learned how to build a proffesional digital presence, comminicate with proper email etiqutte, and stay safe from cyber threats. Additionally, this project helped me understand how platforms like GitHub and HackerRank can support techanical skill development

I created an infographic using canva that explains the concept of digital literacy, highlights useful digital tools and promotes safe internet practices for students

I established my professional digital presence by creating accounts on key platforms:
GitHub - for project portfolio and version control
